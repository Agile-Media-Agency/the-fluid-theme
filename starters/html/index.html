Why Fluid Theme?

Modern CSS that works with your HTML, not against it.

Semantic HTML

Use meaningful elements. No utility class soup. Your HTML stays readable and accessible.

No Build Step

Link one CSS file and go. Works directly in any browser without a bundler or toolchain.

Physics-Based Motion

7 motion presets derived from fluid dynamics. Animations that feel natural, not mechanical.

Dark Mode Built-In

Toggle dark/light with a single attribute. Deep palette colors, not just inverted blacks.

10 Color Palettes

Swap the entire look with one attribute. Each palette includes 6 accent color options.

Accessible by Default

Focus-visible, ARIA-ready components. Native HTML elements — no JS required for core UI.

About This Starter

This is the HTML starter for @fluid-theme/core. It demonstrates the core patterns: semantic HTML, data attributes for variants, CSS custom properties for theming.

Customize the theme by changing attributes on the <html> element: data-theme, data-motion, data-palette, data-accent.
