Built Different

Fluid Theme is pure CSS with a clear philosophy: semantic HTML should be enough.

⬡

Semantic HTML First

Write <button data-variant="outline">, not <div class="btn btn-outline-primary">. Your markup stays clean, accessible, and readable.

◈

Runtime Theming

Switch the entire design personality with one HTML attribute. No JavaScript required, no CSS rebuild — the browser handles it all.

⬟

Complete Presets

Each preset changes fonts, radius, shadows, spacing, background tints, and dark mode surfaces together. A true design system, not a color swap.

◎

Framework Agnostic

React, Vue, Svelte, or plain HTML — import one CSS file and everything works. Components are HTML patterns, not framework components.

◉

Accessibility Built-In

Focus-visible, ARIA-ready, prefers-reduced-motion respected globally. Designed with care for users with disabilities.

◇

CSS Custom Properties

Every design value is a token: var(--space-4), var(--radius-card). Override anything, at any scope, without touching the source.

Up in 30 Seconds

Three ways to use Fluid Theme:

CDN — Zero Setup

<link rel="stylesheet"
  href="https://cdn.jsdelivr.net/npm/@fluid-theme/core/dist/fluid.min.css">

npm — Modern Workflow

npm install @fluid-theme/core

// In your entry file:
import '@fluid-theme/core';

Configure Your Preset

<html
  data-theme="light"       <!-- light | dark | auto -->
  data-preset="flowing"    <!-- still | serene | stream | flowing | cascade | rapids | tsunami -->
  data-palette="fluid"     <!-- fluid | morandi | bold | pastel | earth | mono | 80s | matrix | subtle | tsunami -->
  data-accent="1"          <!-- 1 – 6 -->
>
